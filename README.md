# nodemailer-ses-smtp-demo

## Environment variables
```
# SMTP Settings in AWS SES
SES_SMTP_HOST=""
SES_SMTP_PORT=465
SES_SMTP_USER=""
SES_SMTP_PASS=""

# Email Constants
# Note: If AWS SES is still in SANDBOX, you need to register
# and verify both from and to email addresses
EMAIL_FROM="register.this.email@domain.com"
EMAIL_TO="register.this.other.email@anotherdomain.com"
EMAIL_SUBJECT="TEST FROM AWS SES: Please Ignore"
```
